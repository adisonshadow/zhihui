# 图片编辑功能设计（面向非专业用户）

## 入口与路由

- 主导航「图片编辑」→ `/image-editor`（全屏编辑区，无内容区 padding）。
- 实现：`AppHeader`、`App.tsx` 路由、`ImageEditorPage`。

## 布局

| 区域 | 说明 |
|------|------|
| 顶栏左 | 文件（新建 / 本地打开 / 素材库）、插入图片·形状·文本、AI 生成 |
| 顶栏右 | 项目选择（入库 / 抠图 / 素材库必用）、导出 PNG、保存到素材、缩放滑块 |
| 左侧 | 图层列表与上移/下移快捷按钮 |
| 中间 | Konva 画布：外层棋盘格，内层「文档」矩形（可设背景色） |
| 右侧 300px | 未选中：画布尺寸说明 + 背景色；选中：按类型显示属性 |

## 数据与对象类型

- **文档**：宽、高、背景色（默认白）。
- **图片**：`src`（data URL 为主）、透明度、模糊、一键样式（鲜艳/黑白/冷暖）、去背景（走项目内 `saveAssetFromBase64` + `matteImageAndSave` 替换临时素材）、去白底（`instantTransparencyFromDataUrl`）。
- **形状**：圆角、纯色/渐变、投影、简版毛玻璃（矩形 Blur + 透明度）、子 Tab「文字」（居中字、倒置 180°、模糊、透明）。
- **文本**：内容、字体（系统 `getFonts`）、字号、粗斜、一键样式、描边预设、模糊、透明。

## 交互

- 单击选中；拖拽移动；Transformer 缩放与旋转。
- 右键：删除、置顶/上移/下移/置底（文档 `mousedown` 清空选中）。
- 导出：`Stage` 内文档 `Group.toDataURL`；主进程 `writeBase64File` 写入用户选定路径。
- 保存到素材：`saveAssetFromBase64` → `prop`。

## Electron IPC

- `app:fs:writeBase64File`：导出 PNG。
- `app:fs:readFileAsDataUrl`：打开本机图片。

## AI 生成

- 使用设置中具备「绘图」能力且已配置 API 的模型；逻辑与 `OpenAIImagesProvider` 一致（见 `imageEditorAi.ts`）。

## 依赖

- `konva`、`react-konva`（由用户本地安装）。
