# 素材面板分类方案（面向普通用户）

## 一、分类结构

素材面板分为 **6 个分类**：

| 分类 | 说明 |
|------|------|
| **角色** | 与角色关联的素材：精灵图、元件、透明视频 |
| **布景** | 场景背景、室内外环境：图片、精灵图、元件、视频、透明视频 |
| **道具** | 情景道具：图片、精灵图、元件、视频、透明视频 |
| **特效** | 特效素材：透明视频（暂仅此类） |
| **文字** | 文字素材：文字（暂仅此类） |
| **声音** | 音效与音乐：沿用现有音乐/音效 UI |

---

## 二、Header 统一规范

所有分类的列表 header 统一为 **1 行**：

```
┌─────────────────────────────────────────────────────────────┐
│  [添加 ▼]                                     [模糊搜索 ▼] |
└─────────────────────────────────────────────────────────────┘
```

- **左侧**：添加按钮，点击后弹出 **Dropdown Menu**（各分类下选项不同）
- **右侧**：模糊搜索输入框，带 **Dropdown Menu**（水平展示，Tag 样式）：
  - **角色**：Dropdown 为所有角色名，点击后列表跳转到对应角色
  - **其他分类**：Dropdown 为该分类下的 **Hot Tags**（可配置，暂时写死在代码中），不同分类的 Hot Tags 不同

---

## 三、Card UI 规范

### 3.1 角色、布景、道具、特效、文字（统一 Card）

```
┌─────────────────────────────────┐
│                      [添加 icon] │  ← 右上角：点击添加到时间线主层
│                                 │
│           封面图片/视频          │
│                                 │
│ [精灵图/元件/透明视频]           │  ← 左下角：素材类型标签
└─────────────────────────────────┘
```

- **封面左下角**：素材类型标签（精灵图、元件、透明视频）
- **封面右上角**：添加 icon，点击后添加到时间线主层
- **拖拽**：Card 可按住拖拽到时间线轨道上
- **点击**：点击非添加按钮区域，根据素材类型打开对应素材的预览/编辑面板

### 3.2 声音（沿用现有）

- **声音** 的 Card UI 与之前音乐/音效一致，不做改动

## 四、各分类内容与 Dropdown 配置

### 4.1 角色

| 项目 | 内容 |
|------|------|
| **列表展示** | 按角色分组，将精灵图、元件、透明视频混合在一起列出 |
| **添加按钮 Dropdown** | 精灵图、元件、透明视频 |
| **模糊搜索 Dropdown** | 所有角色名，点击后列表跳转到对应角色 |

### 4.2 布景

| 项目 | 内容 |
|------|------|
| **列表展示** | 按布景分类，将图片、精灵图、元件、视频、透明视频混合在一起列出 |
| **添加按钮 Dropdown** | 图片、精灵图、元件、视频、透明视频 |
| **模糊搜索 Dropdown** | 布景 Hot Tags（写死，如：室内、室外、山水、天空等） |

### 4.3 道具

| 项目 | 内容 |
|------|------|
| **列表展示** | 按道具分类，将图片、精灵图、元件、视频、透明视频混合在一起列出 |
| **添加按钮 Dropdown** | 图片、精灵图、元件、视频、透明视频 |
| **模糊搜索 Dropdown** | 道具 Hot Tags（写死，如：桌椅、马车、武器等） |

### 4.4 特效

| 项目 | 内容 |
|------|------|
| **列表展示** | 与布景/道具类似，仅展示透明视频（暂仅此类） |
| **添加按钮 Dropdown** | 透明视频 |
| **模糊搜索 Dropdown** | 特效 Hot Tags（写死，如：光效、粒子、转场等） |

### 4.5 文字

| 项目 | 内容 |
|------|------|
| **列表展示** | 与布景/道具类似，仅展示文字（暂仅此类） |
| **添加按钮 Dropdown** | 文字 |
| **模糊搜索 Dropdown** | 文字 Hot Tags（写死，如：标题、对白、旁白等） |

### 4.6 声音

| 项目 | 内容 |
|------|------|
| **列表展示** | 按音效/音乐分组，沿用现有 UI |
| **添加按钮 Dropdown** | 音效、音乐（或沿用现有上传入口） |
| **模糊搜索 Dropdown** | 声音 Hot Tags（写死） |

## 五、Hot Tags 配置（暂写死）

| 分类 | Hot Tags 示例 |
|------|---------------|
| 布景 | 室内、室外、山水、天空、街道、建筑 |
| 道具 | 桌椅、马车、武器、服饰、生活用品 |
| 特效 | 光效、粒子、转场、烟雾、火焰 |
| 文字 | 标题、对白、旁白、字幕 |
| 声音 | 音效、BGM、环境音 |

*后续可改为配置项，支持项目级或全局自定义*
