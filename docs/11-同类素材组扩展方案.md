# 同类素材组（列表级素材包）扩展方案

## 目标

- 列表中一项可对应多个文件（如「花瓣雨」含多套雨特效）。
- **素材库**与**设计器素材面板**均展示组卡片；**添加**或**拖到时间线**时先弹出 Modal 选择子项（支持预览、设计器内需解析真实 `assetId`）。
- **预览 Drawer**：若已在组内，正文顶部展示 `AssetBundlePreviewNav`（上一条 / 下一条 / 名称 / 删除，删除为全局删素材）；若未入组，在「保存」旁提供「添加同类素材」（选文件 → 导入 → `addSimilarAssetToBundle`）。透明视频走 `saveTransparentVideoAsset`（自动抠色等默认参数）。

## 数据与主进程

- 表：`asset_bundles`、`asset_bundle_members`；API 见 `electron/main/projectDb.ts`（`getAssetBundleForAsset`、`addSimilarAssetToBundle`、`getBundledAssetIds` 等）。
- IPC：`electron/main/index.ts` + `electron/preload/index.ts`；渲染端类型：`src/vite-env.d.ts`。

## 前端要点

- 类型与 Modal：`src/types/assetBundle.ts`、`AssetBundlePickModal`、`AssetBundlePreviewNav`。
- 设计器：`DesignerTab` 挂载 `AssetBundlePickModal`，`requestAssetBundlePick` 传给 `TimelinePanel` 与 `AssetBrowsePanel`；预览走 `AssetBrowsePanelHandle.openBundleMemberPreview`。
- 时间线：拖拽 `assetBundleId` 时解析子素材后再建块（`TimelinePanel`）。
- 预览：`ImagePreviewDrawer` / `VideoPreviewDrawer` / `AudioPreviewDrawer` 集成组导航与添加同类；需传入 `saveAssetFromFile`、`openFileDialog`（视频透明类型另用 `saveTransparentVideoAsset`）。

## 允许归入组内的素材类型

与 `BUNDLE_ALLOWED_TYPES` 一致：`image`、`video`、`transparent_video`、`sfx`、`music`。  
**同一组内**所有子项的 `assets_index.type` 须**完全相同**（例如 `video` 与 `transparent_video` 不可混组）。
